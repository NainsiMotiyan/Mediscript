# Mediscript

**Mediscript** is a full-stack web application designed as a **prescription management and patient care assistant**.  
It helps users **scan, view, manage, and organize medical prescriptions**, and provides a clean interface for doctors, clinics, and patients to store, retrieve, and review prescription records.
